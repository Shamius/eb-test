# Онлайн-тест: работа с экономическим покупателем (Миллер–Хайман)

Автономный тест на HTML/CSS/JS (без библиотек). Идеален для GitHub Pages: просто выложи файлы — и всё заработает.

## Файлы
- `index.html` — страница теста  
- `styles.css` — оформление  
- `data.js` — банк вопросов (редактируй его)  
- `app.js` — логика (перемешивание, подсчёт, обратная связь)
